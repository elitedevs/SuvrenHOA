## About SuvrenHOA

SuvrenHOA is a modern, blockchain-based homeowner association platform that brings transparency, efficiency, and community engagement to neighborhood governance. Built on secure blockchain technology, it ensures all transactions, votes, and records are permanent and verifiable.
